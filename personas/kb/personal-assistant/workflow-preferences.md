# Workflow Preferences

## Communication Style

- Prefer concise, bullet-point summaries
- Executive briefing format for daily updates
- Include actionable recommendations

## Priority Framework

1. **Blocking others** - Items where people are waiting
2. **Due dates** - Imminent deadlines
3. **Quick wins** - Low effort, high impact
4. **Deep work** - Larger strategic items

## Tools & Integrations

### Primary Tools
- **GitHub**: Code review, PR management
- **Linear**: Issue tracking, sprint planning
- **Berry**: Memory and context

### Notification Preferences
- Morning briefing at 9 AM
- Critical issues only during focus time
- Weekly summary on Fridays
